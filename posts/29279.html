Python 實現桑基圖

Python python 2019/12/10

接續上次 RFM 用戶分析模型，透過桑基圖繪出用戶三個時間點分群的變化，用來觀察整體訪客經營是否符合預期中的變化。

桑基圖 (Sankey Diagram)

桑基圖是一種流程圖，其資訊包含著流量的大小、方向及變化狀況，套用在不同階段的客戶分群上，可以簡單看出訪客的變化狀況。

客戶分群意義

接續 RFM 用戶分析模型將訪客分成 8 組。

Recency 消費時間 Frequency 頻率 Monetary 金額 客戶分群
近 高 高 高價值，VIP 客戶(high_vip)
遠 高 高 高價值，流失客戶(high_passive)
近 低 高 高價值，新客戶(high_new)
遠 低 高 高價值，淺在客戶(high_risk)
近 高 低 一般價值，VIP 客戶(low_vip)
遠 高 低 一般價值，流失客戶(low_passive)
近 低 低 一般價值，新客戶(low_new)
遠 低 低 一般價值，淺在客戶(low_risk)

實作

分群步驟參照前一篇 RFM 用戶分析模型。

RFM 分群

資料日期範圍為 2010-12-01~2011-12-09，將日期分成三個階段，分別做 RFM 分群，同時給予 R,F,M 三個 1 或 2 的分數，再依照時間區間與分群結果定義分群名稱。

  • part1: 2010-12-01 ~ 2011-03-31
  • part2: 2011-04-01 ~ 2011-07-31
  • part3: 2011-08-01 ~ 2011-12-09

Merge

合併三個階段的資料表。

繪圖準備

節點 nodes

# 將三個階段的節點分開
nodes_1 = [l + '_1' for l in rfm_df['level_part1'].unique()]
nodes_2 = [l + '_2' for l in rfm_df['level_part2'].unique()]
nodes_3 = [l + '_3' for l in rfm_df['level_part3'].unique()]
nodes = [*nodes_1, *nodes_2, *nodes_3]

流動方向與值 (source, target, value)

  • source 流動起點(節點的index)
  • target 流動終點(節點的index)
  • value 流動的量
# 合併階段 1to2 與 2to3 的變化
rfm_df_select = rfm_df[['user', 'level_part1', 'level_part2', 'level_part3']]
rfm_df['level_change_1to2'] = rfm_df['level_part1'] + '_1>' + rfm_df['level_part2'] + '_2'
rfm_df['level_change_2to3'] = rfm_df['level_part2'] + '_2>' + rfm_df['level_part3'] + '_3'

# 計算變化量
source = []
target = []
value = []
for idx, val in rfm_df['level_change_1to2'].value_counts().iteritems():
    source.append(nodes.index(idx.split('>')[0]))
    target.append(nodes.index(idx.split('>')[1]))
    value.append(val)
for idx, val in rfm_df['level_change_2to3'].value_counts().iteritems():
    source.append(nodes.index(idx.split('>')[0]))
    target.append(nodes.index(idx.split('>')[1]))
    value.append(val)

繪圖

不過資料變化沒有很大。其他桑基圖的使用參考。

fig = go.Figure(data=[go.Sankey(
    node = dict(
      pad = 20,
      thickness = 20,
      line = dict(color = "black", width = 0.5),
      label = nodes,
      color = "blue"
    ),
    link = dict(
      source = source,
      target = target,
      value = value
  ))])

fig.update_layout(title_text="訪客生命週期可視化", font_size=10)
fig.show()
