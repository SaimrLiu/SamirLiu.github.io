Markdown 簡易手冊

• 2019年11月09日

標題當您想要將一段文字標示為標題的時候，請使用這個語法。請在行首輸入 # 符號，並在後面加上一個空白後，再輸入標題文字。標題有分成 6 個層級，# 的數量愈多，層級越低。 您可以按照範例輸入看看： # 第一層級標題 ## 第一層級標題...
